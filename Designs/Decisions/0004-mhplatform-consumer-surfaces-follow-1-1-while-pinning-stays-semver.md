# ADR 0004: MHPlatform Consumer Surfaces Follow 1.1 While Pinning Stays Semver

- Date: 2026-03-23
- Status: Superseded
- Superseded by: ADR 0005

## Note

This ADR recorded the short-lived MHPlatform 1.1 adoption shape for Fluel:
`MHAppRuntime` as the app base product together with a repository-specific
explanation around the project requirement.

MHPlatform 1.2 replaced that framing with the `MHPlatform` / `MHPlatformCore`
default pillar model that Fluel now follows. See ADR 0005 for the current
consumer-path and package-governance contract.
